Class Mcap0StreamReader

A streaming reader for MCAP files.

Usage example:

const reader = new Mcap0StreamReader();
stream.on("data", (data) => {
  try {
    reader.append(data);
    for (let record; (record = reader.nextRecord()); ) {
      // process available records
    }
  } catch (e) {
    // handle errors
  }
});
