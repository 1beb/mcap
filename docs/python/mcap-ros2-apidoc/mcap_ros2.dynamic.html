mcap_ros2.dynamic module¶

ROS2 message definition parsing and message deserialization.

mcap_ros2.dynamic.generate_dynamic(schema_name: str, schema_text: str) → Dict[str, Callable[[bytes], SimpleNamespace]][source]¶

Convert a ROS2 concatenated message definition into a dictionary of message parsers.

Modeled after the generate_dynamic function in ROS1 genpy.dynamic.

Parameters:

  • schema_name – The name of the schema defined in schema_text.

  • schema_text – The schema text to use for deserializing the message payload.

Returns:

A dictionary containing the generated message.

mcap_ros2.dynamic.read_message(schema_name: str, msgdefs: Dict[str, MessageSpecification], data: bytes) → SimpleNamespace[source]¶

Deserialize a ROS2 message from bytes.

Parameters:

  • schema_name – The name of the schema to use for deserializing the message payload. This key must exist in the msgdefs dictionary

  • msgdefs – A dictionary containing the message definitions for the top-level message and any nested messages.

  • data – The message payload to deserialize.

Returns:

The deserialized message.
