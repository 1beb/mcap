mcap_ros1.decoder module¶

mcap_ros1.decoder.decode_ros1_messages(message_iterator: Iterator[Tuple[Schema, Channel, Message]], ignore_non_ros1_messages: bool = False) → Iterator[Tuple[str, Any, int]][source]¶

takes a stream of messages from a McapReader, and automatically parses the ROS 1 messages using the definitions in the MCAP.

Parameters:

  • message_iterator – an iterator of Schema, Channel, and Message records. Use mcap.mcap0.reader.McapReader.iter_messages() to create this.

  • ignore_non_ros1_messages – if True, ignores non-ros1 messages in the MCAP rather than raising an exception.

Returns:

an iterator of (topic, ros1_message, log_time) tuples. Timestamps are provided as a nanosecond unix timestamp.
